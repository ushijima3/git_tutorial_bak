git log
gitのlogを表示する

git log --oneline
gitのlogを1行で表示する

git log -p index.html
ふぁいるの変更差分を表示する
-p ファイル名を指定することでそのファイルだけの変更差分を表示することが出来る

git rm ファイル名
ファイルを削除

git rm -r ディレクトリ名
ディレクトリを削除

git rm --cached ファイル名
ファイルを残したいとき
rmはremoveの略

git mv 旧ファイル 新ファイル
ファイル名の変更を記録

#以下のコメントと同じ
mv 旧ファイル 新ファイル
git rm 旧ファイル
git add 新ファイル

git remote add origin url
originというショートカットでurlのリモートリポジトリを登録

git remote add origin 

git push リモート名 ブランチ名
ローカルリポジトリの内容をリモートリポジトリに送ることをpushという

コマンドにエイリアスをつける
コマンド入力を省略できる

git config --global alias.ci commit
git config --global alias.st status
git config --global alias.br branch
git config --global alias.co checkout

ver管理しないファイルをgitの管理から外す
.gitignoreファイルに指定する

#指定したファイルを除外
index.html

# ルートディレクトリを指定
/root.html

#ディレクトリ以下を除外
dir /

# /以外の文字列にマッチ [*]
/*/*.css
